    Knowledge - Vertical vs Horizontal scaling

    Vertical scaling: Increase the power of machines. E.g. upgrade from 4 vCPU to 8 vCPU --> Scaling Up

    Horizontal scaling: Add more machines. E.g. 3 web servers instead of 1 --> Scaling Out

    — Simon (@simonholdorf) October 2, 2021
